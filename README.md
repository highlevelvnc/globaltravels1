# Global Travel Preview

Pacote preview do site com as páginas principais e um simulador reconstruído.

## O que tem aqui
- `index.html`
- `nossos-cursos.html`
- `trabalhar-morar-fora.html`
- `simulador.html`
- `assets/site.css`
- `assets/site.js`
- `assets/simulador.js`

## Observação importante
Este pacote foi montado para você testar a experiência completa e subir em um novo repositório. As imagens de marca estão apontando para os arquivos públicos do repositório original no GitHub, então o projeto abre sem precisar que eu te envie os binários daqui.

## Como testar
1. Descompacte o arquivo.
2. Abra `index.html` no navegador.
3. Navegue até `simulador.html` para testar o fluxo.
4. Se quiser subir no GitHub Pages, envie todos os arquivos mantendo a mesma estrutura.
